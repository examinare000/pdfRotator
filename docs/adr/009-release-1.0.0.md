# ADR-009: v1.0.0 リリース

## ステータス
採用済み（2025-12-23）

## 背景
- PDF.js worker のバージョン不一致が原因でPDF読み込みが失敗する事象があった。
- 現在の修正内容を最初のリリース（v1.0.0）として扱う方針に変更した。

## 決定
- 現在のリリース内容を `main` にマージし、タグ `v1.0.0` を付与する。
- 既存のリリースタグは整理し、`v1.0.0` を正とする。
- 配布パッケージは `scripts/package-win.ps1` により生成する。

## 影響範囲
- `main` リリース履歴と配布物の更新。
